js原型的理解

Js所有的函数都有一个prototype属性，在js中可以记为：_proto_,这个属性引用了一个对象，即原型对象， 也简称原型,按照字面意思来理解，那么prototype就是通过调用构造函数而创建的那个对象实例的原型对象。

只要创建一个新函数，就会根据一组特定的规则为该函数创造一个prototype的属性，指向函数的原型对象。默认情况下，所有原型对象都会自动获得一个constructor属性，这个属性包含一个指向prototype属性所在函数的指针， 而通过这个构造函数，可以继续为原型对象添加其他属性和方法。

创建自定义的构造函数之后，其原型对象默认只会取得constructor属性，其他方法，将从object继承得。调用构造函数创建的第一个实例后，该实例的内部将包含一个指针 （__proto__ 属性，指向构造函数的原型对象

每个函数内部的__proto__指向构造他们的函数的prototype.

function A(x){ this.x=x; }

var obj=new A(1);
