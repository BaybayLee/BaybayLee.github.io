js array的splice和slice方法理解

Array.slice()

Array.slice()方法返回指定数组的一个片段或者子数组，返回的是包含第一个参数的指定位置和多有到，但不包含第二个参数指定位置之间的所有数组元素。如果只指定一个，那么从，开始位置到数组 结束位置的所有元素。如果参数中出现负数，它表示相对于数组中的最有一个元素的位置。例如，-1指定了最后一个元素，-3指定了倒数第三个元素。slice()不会修改调用的数组。

var a=[1,2,3,4,5];

a.slice(0,3);//返回［１，２，３］

a.slice(3);//[4,5]

a.slice(1,-1);//[2,3,4]

a.slice(-3,-2);//[3]

Array.splice()

Array.splice()是在数组中插入或者删除元素的通用方法， 不同于slice() 和concat(),splice()方法会修改调用的数组

splice()能够从数组中删除元素、插入元素或同时完成这两种操作。splice()的第一个参数指定了插入或删除的起始位置，第二个参数指定了从数组中删除的 元素个数，如果省略第二个参数，从起始点，到结尾的所有元素都会被杉树。splice()返回一个由删除元素组成的数组如果没有删除数组，就返回一个空数组

var a=[1,2,3,4,5,6,7,8]

a.splice(4); //返回[5,6,7,8]，a是[1,2,3,4]

a.splice(1,2) //返回[2,3]，a是[1,4]

a.splice(1,1);//返回[4]，a是[1]

slice()的前两个参数指定了需要删除的数组元素，紧随其后的任意个数的参数指定了需要插入到数组中的元素，从第一个参数知道那个的位置开始插入

var a=[1,2,3,4,5,6,7,8]

a.splice(2,0,'a','b');//返回[]，a是[1,2,'a','b',3,4,5,6,7,8]

a.splice(2,2,[1,2],3);//返回['a','b']，a是[1,2,[1,2],3,3,4,5,6,7,8]

forEach()

forEach()方法从头值尾遍历数组，为每个元素调用指定的函数。传递的函数作为forEach的第一个参数，然后forEach()中使用三个参数调用该函数：数组元素，元素的索引和数组本身。如果只关心数组元素的值，可以编写只有一个参数的的函数，其他参数将被忽略

var data=[1,2,3,4,5]

data.forEach(function(v,i,a){a[i]=v+1;//[2,3,4,5,6],每个元素加1
